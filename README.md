Inventory Management System

Project Overview

Inventory Management System is a full-stack web application developed using Node.js, Express.js, MySQL, Next.js, and Tailwind CSS.

The system allows administrators to manage products, inventory stock, customer orders, and user authentication.

---

Features

Authentication

- User Login
- JWT Authentication
- Protected Routes
- Logout Functionality

Dashboard

- Total Users
- Total Products
- Total Orders
- Total Revenue
- Pending Orders

Product Management

- Add Product
- Edit Product
- Delete Product
- Add Stock
- Reduce Stock
- Low Stock Alerts
- Product Search
- Product Filters
- Pagination

Order Management

- Place Orders
- Cancel Orders
- Restore Stock on Cancellation
- Order Status Tracking
